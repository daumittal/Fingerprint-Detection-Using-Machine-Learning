# Fingerprint-Detection-Using-Machine-Learning

In this project, we aim to explore authorship attribution using unique traits in an author’s written works. Our dataset consists of songs from renowned songwriters, including titles, lyrics, and author details. We'll develop a model for accurately attributing authorship, with applications in plagiarism detection, literary analysis, and more.
First, we'll load the dataset and language model for text processing. Then, we'll preprocess the text to reduce noise and extract relevant linguistic features like word length distribution and frequency. We'll create a training corpus and employ Burrows's Delta for authorship attribution.
Ultimately, we'll construct a high-accuracy model for authorship attribution and investigate its potential for analyzing an author’s evolving style over time
